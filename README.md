# 🏏 Vaibhav Sooryavanshi IPL Performance Analysis | Power BI

## 📌 Project Overview

This project analyzes the IPL batting performance of **Vaibhav Sooryavanshi** using Power BI. The dashboard provides interactive insights into batting statistics, opponent-wise performance, boundary analysis, and year-wise comparisons.

---

## 🎯 Objectives

* Analyze overall batting performance.
* Compare IPL 2025 and IPL 2026 statistics.
* Study performance against different opponents.
* Visualize boundary contribution and dismissal patterns.
* Create an interactive dashboard using Power BI.

---

## 📊 Key Performance Indicators (KPIs)

* Total Runs
* Average Score
* Strike Rate
* Highest Score
* Total Fours
* Total Sixes
* Boundary Percentage
* Opponent-wise Runs
* Year-wise Comparison
